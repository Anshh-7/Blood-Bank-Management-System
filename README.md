# Blood Bank Management System

A Python-based Blood Bank Management System designed to simplify the management of blood donations, blood requests, donor information, and blood inventory. The application features a user-friendly graphical interface built with Tkinter and provides efficient tracking of blood stock levels.

## Features

* Donor Registration
* Blood Request Management
* Blood Inventory Tracking
* User-Friendly GUI
* Data Storage and Retrieval
* Real-Time Stock Updates

## Technologies Used

* Python
* Tkinter
* File Handling
* Object-Oriented Programming (OOP)

This project was developed as a learning project to apply Python programming concepts and create a practical healthcare management solution.
